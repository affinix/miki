<h1 align="center">Miki</h1>
<p align="center">Discord bot for Anime@UTS.</p>
---
## Developing
Requires Deno 2.1.4 or newer.
```bash
git clone https://github.com/affinix/miki.git
cd miki
deno install
```
Then, copy .env.example to .env and fill in required values, and run `deno task dev` to start development enviroment.
