<h1 align="center">miki</h1>
<p align="center">Discord bot for <a href="https://utsanime.net/">Anime@UTS.</a></p>

## Developing
Requires [Deno](https://deno.com/) 2.1.4 or newer.
```bash
git clone https://github.com/affinix/miki.git
cd miki
deno install
```
Copy `.env.example` to `.env` and fill in required values. Then:
```bash
deno task init
deno task dev
```
